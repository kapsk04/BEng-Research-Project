# AI-enabled Automatic Reformatting of 3D Cardiac MR Images

This repository contains the code used for my project on AI-enabled automatic reformatting of 3D cardiac MR images.

## Contents

- Synthetic experiments for plane heatmap prediction
- Clinical DICOM preprocessing
- 3D U-Net training for LGE volume to cine plane heatmap prediction
- Evaluation using heatmap L1/MSE, plane angle error, and reformatted slice/stack RMSE
